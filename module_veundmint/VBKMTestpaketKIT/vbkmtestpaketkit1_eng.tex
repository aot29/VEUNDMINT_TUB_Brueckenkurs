\begin{MExercise}
Rewrite the absolute value expression $|2x-1|-3x$ as a case analysis with two expressions that do not contain 
any absolute values.\\
Answer: $|2x-1|-3x$ = \MLSimplifyQuestion{40}{abs(2*x-1)-3*x}{8}{x}{3}{128}{TTXT11}.\\
\MInputHint{A short version of a case analysis is entered in the form \texttt{if(condition,value1,value2)}.}
\ \\ \ \\
\begin{MHint}{What to do here}
In a mathematical case analysis, absolute values are defined by different cases. For example, the absolute value 
expression $|x-1|$ can be written as:
$$
|x-1| \;=\; \MCases{x-1 & \text{if}\;x\geq 1 \\ -x+1 & \text{if}\; x<1}
$$
\MInputHint{For example, this expression can be entered in the input box as \texttt{if(x>=1;x-1;-x+1)}. Please note that there are several correct ways 
to write these expressions.}
\end{MHint}
\end{MExercise}
