Aufgabe 1.4.11  
Berechnen Sie die folgenden Potenzen:

    a) 53


    53 =125


    b) (-1 )1001


    (-1 )1001 =-1


    c) ((-3 )2 )3


    ((-3 )2 )3 = 93 =729


    d) (- 3 5 )4


    (- 3 5 )4 = 34 54 = 81 625


    e) ( 2-2 )-3


    ( 2-2 )-3 = 2(-2)·(-3) = 26 =64


    f) (- 1 2 )0


    (- 1 2 )0 =1



Aufgabe 1.4.12  
Vereinfachen Sie die folgenden Ausdrücke mit den Potenzregeln:

    a) 33 · 35 · 3-1


    33 · 35 · 3-1 = 33+5-1 = 37


    b) 42 · 32


    42 · 32 =(4·3 )2 = 122


    c) (-2 )7 (-2 )5


    (-2 )7 (-2 )5 =(-2 )7-5 =(-2 )2


    d) 62 · 3-2


    62 · 3-2 = 62 · ( 1 3 )2 = (6· 1 3 )2 = 22


    e) 643 83


    643 83 = ( 64 8 )3 = 83


    f) ( 3 4 ) 1 3 · ( 3 4 ) 2 3


    ( 3 4 ) 1 3 · ( 3 4 ) 2 3 = ( 3 4 ) 1 3 + 2 3   =   3 4



Aufgabe 1.4.13  
Berechnen Sie die folgenden Wurzeln:

    a) (35)5


    (35)5 =( 35 ) 1 5 = 35· 1 5 =3


    b) 2564


    2564= ( 28 ) 1 4 = 22 =4


    c) 33·93


    33·93=3·3·33=3


    d) 3433


    3433= 73 3=7


    e) 276


    276= 33 3=3

    a) 33 · 63 23 · 43


    33 · 63 23 · 43 = 36 · 23 23 · 43 = 36 43

    4235= (( 4 1 2 ) 1 3 ) 1 5 = 4 1 2 · 1 3 · 1 5 =430=215

    32 · 9-3 · 276 = 32+2·(-3)+3·6 = 314

    a3 5 a5 4 = a 3 5 - 5 4 = a 12-25 20 = a- 13 20
