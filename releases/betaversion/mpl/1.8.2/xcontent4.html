Aufgabe 8.2.17  
Berechnen Sie die Integrale:
  1. ∫-3 2 (1+6 x2 -4x)dx  =  .

  2. ∫1 9 5 4x dx  =  .



Der Integrand f(x)=1+6 x2 -4x=6 x2 -4x+1 ist ein Polynom, sodass F(x)=2 x3 -2 x2 +x eine Stammfunktion ist. Mit dem Hauptsatz folgt

∫0 1 (1+6 x2 -4x)dx= [2 x3 -2 x2 +x]-3 2 =2(8-4)+2-(2(-27-9)-3)=85.

Beim zweiten Aufgabenteil ist der Integrand f(x)= 5 4x = 5 2 x-1/2 ein Produkt einer Wurzelfunktion mit einem konstanten Faktor. Damit ist F(x)=5 x1/2 =5x eine Stammfunktion von f. Mit dem Hauptsatz folgt

∫1 9 5 4x dx= [5x]1 9 =5(3-1)=10.

Aufgabe 8.2.19  
Berechnen Sie die Integrale
  1. ∫0 3 (2x-1)dx  =  .

  2. ∫-3 0 (1-2x)dx  =  .



Der Integrand f(x)=2x-1 ist eine Polynomfunktion. Damit ist F(x)= x2 -x eine Stammfunktion von f. Mit dem Hauptsatz folgt

∫0 3 (2x-1)dx= [ x2 -x]0 3 =9-3-0=6.

Beim zweiten Aufgabenteil ist der Integrand f(x)=1-2x eine Polynomfunktion. Damit ist F(x)=x- x2 eine Stammfunktion von f. Mit dem Hauptsatz folgt

∫-3 0 (1-2x)dx= [x- x2 ]-3 0 =0-(-3-9)=12.







Aufgabe 8.2.20  
Berechnen Sie das Integral

∫π 3π ( 3π x2 -4sin(x))dx  =  .

Zum Integranden f(x)= 3π x2 -4sin(x) ist F(x)=- 3π x +4cos(x) eine Stammfunktion von f. Mit dem Hauptsatz folgt

∫π 3π ( 3π x2 -4sin(x))dx= [- 3π x +4cos(x)]π 3π =(-1-4)-(-3-4)=2.

Die Funktionen sin und cos haben die Eigenschaft, dass das Integral über eine Periode gleich Null ist. Für andere periodische Funktionen wie zum Beispiel |sin(x)| können sich für das Integral über eine Periode auch Werte ungleich Null ergeben.
